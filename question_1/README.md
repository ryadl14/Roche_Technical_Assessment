# Q1: R Package — descriptiveStats

Implements six descriptive statistics functions as a well-structured 
R package with full documentation and tests.

## Functions
- calc_mean(), calc_median(), calc_mode()
- calc_q1(), calc_q3(), calc_iqr()

All functions handle edge cases: empty vectors, non-numeric inputs, 
and NA values. calc_mode() handles ties and the no-mode case.

## Structure

```
descriptiveStats/
├── R/descriptive_stats.R   <- all six functions with Roxygen2 docs
├── tests/testthat/         <- 19 passing tests
├── man/                    <- auto-generated by roxygen2
├── DESCRIPTION
└── NAMESPACE
```

## How to Run
```
devtools::install("question_1/descriptiveStats")
library(descriptiveStats)
devtools::test()            # run tests
devtools::check()           # full package check
```
